# analisis_de_datos_simple
analisis de "Superstore Sales Dataset"

el objetivo de este analisis sera poner aprueba los conocimientos que he estado adquiriendo sobre estadisticas. los objetivos me los dio chat gpt:

🔥 ENTREGABLES OBLIGATORIOS
1️⃣ Limpieza de datos

Revisar valores nulos

Revisar tipos de datos

Convertir fechas correctamente

Verificar duplicados

Explicar cualquier decisión de limpieza

Si no hay problemas, también debes decirlo.

2️⃣ Análisis Univariado

Analiza individualmente:

Ventas

Ganancia

Descuento

Cantidad

Debes:

Mostrar distribución

Detectar outliers

Interpretar qué significan esos outliers

3️⃣ Análisis por Categoría y Subcategoría

Responde:

¿Qué categoría vende más?

¿Cuál deja más ganancia?

¿Hay categorías con muchas ventas pero baja ganancia?

Ranking de subcategorías por ventas y por ganancia

Margen de ganancia por categoría

Interpretación obligatoria.

4️⃣ Análisis Geográfico

Ventas por región

Ganancia por región

Región más rentable

Región con peor desempeño

¿Hay regiones con alta venta pero baja rentabilidad?

5️⃣ Análisis Temporal

Ventas por mes

Ganancia por mes

Tendencia general (crece, decrece, estable)

Estacionalidad

Mejor y peor mes

No solo gráfica. Explicación.

7️⃣ Correlaciones

Haz matriz de correlación entre:

Ventas

Ganancia

Cantidad

Descuento

Interpreta cada relación.
No solo pongas la matriz.
